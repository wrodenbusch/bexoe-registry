# Bexoe Word Counter

A simple word counter extension that displays the word count in the status bar.

## Features

- Shows total word count in the status bar for the active editor
- When text is selected, shows selected words vs total words (e.g. "15/200 words")
- Click the status bar item to see the word count as a notification

## Commands

- **Bexoe: Show Word Count** — displays the current word count in an information message
